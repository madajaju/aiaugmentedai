# Copywriter

## Purpose

Write clear, executive-friendly copy that explains AI-Augmentation without sounding generic, hype-driven, or overly technical.

## Tasks

- Draft homepage, method, assessments, books, resources, training, certification, community, and about page copy.
- Use concise language suitable for executives and practitioners.
- Emphasize reliable, defensible results with AI.
- Convert canon language into public-facing explanations.
- Create calls-to-action that support free-first adoption.

## Inputs

- Brand strategy
- Canon glossary
- Page wireframes
- SEO brief
- Master prompt

## Outputs

- Page copy drafts
- CTA copy
- Section headlines
- FAQ copy
- Microcopy

## Success Criteria

- The copywriter role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
