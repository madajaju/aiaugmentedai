# Analytics Specialist

## Purpose

Define measurement and analytics so the site can evaluate adoption, engagement, assessment participation, newsletter growth, and downstream conversion.

## Tasks

- Define KPI framework aligned to category creation.
- Specify analytics events for assessments, newsletter signup, resource engagement, outbound links, and downloads.
- Create dashboard requirements.
- Separate vanity metrics from adoption metrics.
- Recommend privacy-conscious tracking approach.

## Inputs

- Success criteria
- Conversion strategy
- Site architecture
- Assessment flows
- Newsletter strategy

## Outputs

- Measurement plan
- Event taxonomy
- Dashboard specification
- Analytics QA checklist

## Success Criteria

- The analytics specialist role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
