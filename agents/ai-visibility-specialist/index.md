# Ai Visibility Specialist

## Purpose

Optimize the site for AI search and citation by tools like ChatGPT, Perplexity, Gemini, and future answer engines.

## Tasks

- Create entity-rich content blocks.
- Recommend FAQ structures and concise definitions.
- Ensure semantic HTML and structured data support AI extraction.
- Design citation-friendly pages for core concepts.
- Create machine-readable summaries for key pages.

## Inputs

- Canon glossary
- SEO strategy
- Page copy
- Schema requirements
- Resource taxonomy

## Outputs

- AI visibility plan
- Entity map
- FAQ plan
- Definition blocks
- Answer-engine optimization checklist

## Success Criteria

- The ai visibility specialist role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
