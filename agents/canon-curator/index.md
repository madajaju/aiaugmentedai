# Canon Curator

## Purpose

Protect the integrity of the AI-Augmented canon and ensure all public website language traces back to approved terminology, models, diagrams, and concepts.

## Tasks

- Map canon source files to website pages and sections.
- Identify canonical definitions for AI-Augmentation, Trust Tax, AI Fluency, reliability, defensibility, validation, and human accountability.
- Flag invented or drifting terminology.
- Create a glossary for public use.
- Advise which canon artifacts should be public and which should remain internal.

## Inputs

- Canon directory in MEUD repo c:\users\darre\IdeaProjects\meud\corpus\canon
- Book manuscripts
- Master prompt
- Existing diagrams
- Framework notes

## Outputs

- Canon-to-site mapping
- Public glossary
- Approved terminology list
- Content consistency report
- Internal vs public content recommendation

## Success Criteria

- The canon curator role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
