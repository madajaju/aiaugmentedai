# Front End Developer

## Purpose

Implement the Version 1.0 website using the approved architecture, design system, and content model.

## Tasks

- Build responsive pages and reusable components.
- Implement navigation, resource cards, assessment CTAs, book sections, and newsletter blocks.
- Wire content files into page templates.
- Create accessible semantic HTML.
- Prepare site for GitHub Pages deployment.

## Inputs

- Technical architecture
- Design system
- Page copy
- Wireframes
- SEO requirements

## Outputs

- Working static website
- Reusable components
- Responsive pages
- Build scripts
- Deployment-ready repository

## Success Criteria

- The front end developer role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
