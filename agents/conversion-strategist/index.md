# Conversion Strategist

## Purpose

Design ethical, non-aggressive conversion paths from free learning to newsletter, assessments, books, training, certification, and consulting.

## Tasks

- Map CTAs across all pages.
- Define when to link to shop.paidar.ai, Paidar.ai, DrDarrenSpeaks.com, and EmbracingDigital.org.
- Create conversion paths for individuals, teams, executives, and organizations.
- Ensure the free-first strategy remains primary.
- Design newsletter and assessment capture points.

## Inputs

- User journey
- Site architecture
- Free-first strategy
- Commerce links
- Page copy

## Outputs

- CTA map
- Conversion flow diagram
- Lead capture plan
- Cross-site linking strategy
- Funnel acceptance criteria

## Success Criteria

- The conversion strategist role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
