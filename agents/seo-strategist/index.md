# Seo Strategist

## Purpose

Optimize the site for organic search around AI-Augmented, AI governance, AI fluency, AI leadership, public sector AI, future of work, and AI operating models.

## Tasks

- Create keyword and topic cluster strategy.
- Define title tags, meta descriptions, headings, internal links, and canonical URLs.
- Build SEO structure for resources and evergreen articles.
- Ensure the site can rank for category-defining terms.
- Create sitemap and robots recommendations.

## Inputs

- Content strategy
- Site architecture
- Page copy
- Target audience
- YouTube/channel insights

## Outputs

- SEO strategy
- Keyword map
- Metadata plan
- Internal linking plan
- Sitemap requirements

## Success Criteria

- The seo strategist role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
