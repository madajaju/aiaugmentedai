# Ux Designer

## Purpose

Design the user experience so visitors quickly understand the movement, find the right path, and are guided toward assessments and community.

## Tasks

- Create wireframes for all core pages.
- Design Individual, Team, and Organization path flows.
- Make the assessment pathway obvious without feeling overly sales-driven.
- Design resource discovery and filtering experiences.
- Ensure mobile-first usability.

## Inputs

- Information architecture
- Brand strategy
- Content drafts
- User journey
- Technical constraints

## Outputs

- Wireframes
- User flows
- CTA placement plan
- Mobile interaction guidance
- UX acceptance criteria

## Success Criteria

- The ux designer role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
