# Performance Engineer

## Purpose

Keep the static site fast, lightweight, and reliable on GitHub Pages and Cloudflare.

## Tasks

- Optimize CSS, JavaScript, images, fonts, and build output.
- Check Core Web Vitals.
- Ensure pages load quickly on mobile.
- Reduce unnecessary dependencies.
- Define caching and Cloudflare configuration guidance.

## Inputs

- Implemented site
- Design assets
- Build workflow
- Hosting configuration

## Outputs

- Performance audit
- Optimization checklist
- Core Web Vitals report
- Hosting/caching recommendations

## Success Criteria

- The performance engineer role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
