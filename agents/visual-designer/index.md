# Visual Designer

## Purpose

Create an enterprise-grade visual identity for AI-Augmented.ai that feels related to the existing ecosystem while standing as its own intellectual property platform.

## Tasks

- Define color palette, typography, spacing, components, icons, and visual motifs.
- Use inspiration from paidar.ai and drdarrenspeaks.com without duplicating them.
- Avoid neon AI clichés and generic startup visuals.
- Design framework diagrams and cards for Individual, Team, Organization paths.
- Create thumbnail-ready visuals for sharing resources.

## Inputs

- Brand strategy
- Existing site styles
- UX wireframes
- Canon diagrams
- Design requirements

## Outputs

- Design system
- Component library
- Visual direction board
- Diagram style guide
- Asset requirements list

## Success Criteria

- The visual designer role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
