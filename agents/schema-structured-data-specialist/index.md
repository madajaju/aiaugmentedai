# Schema Structured Data Specialist

## Purpose

Implement structured data so the site is machine-readable, credible, and discoverable by search engines and AI systems.

## Tasks

- Create JSON-LD for Organization, Person, WebSite, Book, Course, Article, FAQPage, and BreadcrumbList.
- Map schema to each page type.
- Validate schema for correctness.
- Connect Dr. Darren, books, AI-Augmented, Paidar, and media properties as related entities.
- Document how to add schema for future resources.

## Inputs

- Site architecture
- Entity map
- Book data
- Founder bio
- Page templates

## Outputs

- Schema implementation plan
- JSON-LD templates
- Validation checklist
- Entity relationship map

## Success Criteria

- The schema structured data specialist role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
