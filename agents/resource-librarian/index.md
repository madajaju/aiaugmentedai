# Resource Librarian

## Purpose

Organize articles, podcasts, videos, worksheets, templates, and case studies into a scalable library that can grow to hundreds of resources.

## Tasks

- Create metadata model for resources.
- Define categories, tags, topics, audiences, and formats.
- Map existing Embracing Digital and book content into the resource library.
- Design resource index and detail page requirements.
- Define editorial governance for future additions.

## Inputs

- Content inventory
- EmbracingDigital.org
- YouTube playlists
- Canon
- Resource taxonomy

## Outputs

- Resource metadata model
- Resource taxonomy
- Initial resource inventory
- Library page plan
- Editorial governance checklist

## Success Criteria

- The resource librarian role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
