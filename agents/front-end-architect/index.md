# Front End Architect

## Purpose

Define the technical architecture for a static, fast, maintainable, GitHub Pages and Cloudflare-compatible website.

## Tasks

- Design directory structure and component patterns.
- Select static HTML, Tailwind CSS, vanilla JavaScript, and Markdown processing approach.
- Create reusable templates for pages, resources, FAQs, books, assessments, and landing pages.
- Ensure performance, accessibility, SEO, and maintainability.
- Define build and deployment workflow.

## Inputs

- Master prompt
- Design system
- Content model
- Repository constraints
- Deployment target

## Outputs

- Technical architecture
- Directory structure
- Template plan
- Build workflow
- Deployment instructions

## Success Criteria

- The front end architect role produces outputs that directly support the AI-Augmented.ai Version 1.0 launch.
- Outputs align with the free-first strategy and category-creation mission.
- Outputs do not turn AI-Augmented.ai into a generic consulting, speaker, software, or podcast site.
- Outputs reinforce the Individual → Team → Organization journey.
